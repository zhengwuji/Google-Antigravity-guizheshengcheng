# 🤖 Antigravity Rules规则生成 (Google Antigravity 标准生成器)

> **一句话介绍**: 这是一个专门用来“调教” AI 助手的工具。它能生成一份专业的 `.cursorrules` 或 Prompt 文件，让您的 AI 助手瞬间变身 Google 高级工程师。

## ✨ 最新核心功能 (v2.2)

### 1. 💡 大白话解释 (Plain Explanations)

**"这功能有啥用？"**
> 专为非硬核技术人员设计。让您瞬间看懂每一个专业名词，不再被一大堆英文术语劝退。

* **即点即现**: 点击界面上的任何 **角色 (Role)**、**技术栈 (Stack)** 或 **环境 (Environment)** 按钮。
* **通俗易懂**: 按钮下方会立即弹出一个提示框，用最接地气的中文解释它的含义。
  * *例子*: 点击 `React` -> 显示 *"Facebook 出的前端霸主，适合做复杂的大型网页..."*

### 2. 🧠 AI 智能配置 (Smart Config)

**"这功能有啥用？"**
> 您不用再手动一项项勾选了。直接告诉它你想做什么，它自动帮您配好。

* **自然语言交互**: 直接输入 "我要做一个仿淘宝的电商APP"，系统自动为您选好角色、技术栈。
* **智能规则提取**: 输入 "每次写完代码必须自动检查"，它会自动把这句话变成一条强制规则。

### 3. 💾 自动备份 (Auto Backup) **[NEW]**

**"这功能有啥用？"**
> 防止 AI 写代码时发生意外，导致旧代码丢失。

* **规则详情**: *"每次修改后，必须自动将代码备份到桌面的'备份'文件夹，文件名需包含修改说明"*。
* **效果**: 当您把规则喂给 AI 后，它在修改任何文件之前，都会乖乖地先把一份副本存到您桌面的 `备份` 文件夹里。从此告别 "代码改崩了回不去"。

### 4. 📝 中文任务强制 (Chinese Task)

**"这功能有啥用？"**
> 专为中国开发者优化，拒绝 AI 蹦英文。

* **规则详情**: *"任务列表 (Task List) 和思考过程 (Thinking Process) 必须强制使用中文书写"*。
* **效果**: AI 的所有计划、步骤拆解都会用简体中文展示，清晰明了。

### 5. 📋 预设管理 (Save & Load)

* **一键保存**: 把当前的所有配置存到浏览器里。
* **一键加载**: 下次打开，点一下就恢复之前的配置。

### 6. 🛡️ 自动化闭环 & 全自动代理

* **自动化闭环**: AI 写完代码必须自己运行检查和测试，自己修 Bug。
* **全自动代理**: 授权 AI 自行决策，不需要每一步都问您 "我可以改这个文件吗？"。

## 🚀 完整使用说明 (Step-by-Step)

### 第一步：打开应用

在 `dist` 文件夹中找到生成的 `Antigravity Rules规则生成 Setup 1.0.0.exe`，双击安装并运行。

### 第二步：配置您的项目

界面主要分为三块区域：

1. **AI 智能配置区 (顶部紫色框)**
    * 在输入框里写一句话描述您的项目（如：“我要用 Python 做一个股票分析系统”）。
    * 点击 **`自动生成`**。系统会自动为您选好角色、技术栈，并添加相关的规则。

2. **手动选择区 (中间)**
    * **角色 (Role)**: 点选您的身份（如前端、后端）。如果不确定的，点击按钮，下方会有中文解释。
    * **技术栈 (Stack)**: 点选您要用的技术。
    * **环境 (Environment)**: 选择您的部署方式（如 Docker）。

3. **规则库 (右侧)**
    * 这里列出了所有可用的 Antigravity 工程标准。
    * **必选推荐**: 建议勾选 `自动备份`、`中文任务`、`自动化闭环` 这三条核心规则。
    * 点击 `+` 号可以添加某条规则。

### 第三步：生成与导出

配置完成后，看界面底部：

* **`复制 (Copy)`**: 最常用的功能。点击后，所有配置会变成一段 Markdown 文本复制到您的剪贴板。
* **`导出 Markdown`**: 将规则下载为一个 `.md` 文件。

### 第四步：使用规则 (喂给 AI)

将复制的内容粘贴给您的 AI 助手，这一步至关重要：

* **Cursor / VS Code**: 粘贴到项目根目录下的 `.cursorrules` 文件中。
* **ChatGPT / Claude**: 直接发送给对话框 "这是我的项目规则，请严格遵守"。
* **Trae / Windsurf**: 粘贴到 `.windsurfrules` 或对应的全局提示词中。

🎉 **大功告成！**
现在，您的 AI 助手已经“觉醒”了。它会开始用中文回复您，写代码前会自动备份，写完后会自动检查，并且严格遵守 Google 级的工程规范。

## 🛠️ 开发者指南 (编译说明)

如果您修改了源码，需要重新生成 EXE：

1. 确保安装 Node.js。
2. 运行 `pnpm install` 安装依赖。
3. 运行 `pnpm run electron:build` 进行编译。
4. 编译成功后，新版 EXE 会出现在 `dist` 文件夹中。
